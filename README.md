# AI Fitness Market Analysis (Power BI Dashboard)

This project analyzes the AI-driven fitness market using real-world data to understand:
- **Market demand signals** (Google Trends)
- **Company performance** (Peloton quarterly metrics)
- **User engagement behavior** (Fitbit activity data)

## Dashboard Pages
1. **Market Demand (Google Trends)** – search interest trends for fitness and AI fitness topics  
2. **Company Growth (Peloton)** – revenue, subscribers, churn, and financial indicators  
3. **User Engagement (Fitbit)** – real user activity patterns (steps, calories, weekday vs weekend)

## Tools Used
- Power BI
- DAX
- Public datasets (Google Trends, Peloton reports, Fitbit dataset)

## Files in This Repository
- `report/` – Project report (PDF)
- `screenshots/` – Dashboard screenshots, data model, measures
